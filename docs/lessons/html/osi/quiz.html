問題一覧

Q1. OSI参照モデルは全部で何層ありますか？

7層です。（アプリケーション層〜物理層）

Q2. TCP/IPモデルの「インターネット層」は、OSIモデルのどの層に相当しますか？

OSIの第3層（ネットワーク層）に相当します。

Q3. HTTPやDNSはどの層に分類されますか？

アプリケーション層に分類されます。

Q4. データの信頼性を確保するプロトコルは？

TCP（トランスポート層）です。

Q5. Pingコマンドが利用するプロトコルは？

ICMP（インターネット層）を利用します。

Q6. データリンク層で使用されるアドレスは何ですか？

MACアドレスです。

Q7. カプセル化の順序で最も外側（送信時の最後）に付くものは？

フレーム（ネットワークインターフェース層）です。

Q8. プレゼンテーション層（OSI第6層）の主な役割は何ですか？

データの表現：暗号化・復号、圧縮・伸長、文字コード変換など。

Q9. トランスポート層で使われる2つの主要プロトコルは？

TCP と UDP です。

Q10. HTTPとHTTPSの既定ポート番号は？

HTTPは80番、HTTPSは443番です。

Q11. UDPが向いている代表的な用途を1つ挙げてください。

リアルタイム通話・ライブ配信・オンラインゲームなど（遅延に強い用途）。

Q12. TCPの「3ウェイハンドシェイク」は何層の動作ですか？

トランスポート層（TCP）の動作です。

Q13. IP（インターネット層）の主な役割は何ですか？

パケットを宛先まで届けるためのアドレッシングとルーティング。

Q14. 「セグメント」「パケット」「フレーム」はそれぞれ何層のデータ単位（PDU）ですか？

セグメント＝トランスポート層、パケット＝インターネット層、フレーム＝ネットワークIF層。

Q15. スイッチとルータの主な違いは？（層と役割で答えて）

スイッチはL2でMACに基づく転送、ルータはL3でIPに基づく経路制御。

Q16. DNSの役割は何ですか？（一言で）

名前解決（ドメイン名 ⇔ IPアドレスの対応付け）。

Q17. サブネットマスクの目的は何ですか？

同一ネットワークの範囲を決める（ネットワーク部とホスト部の境界を示す）。

Q18. MTUを超える大きさのIPパケットはどう扱われますか？

IPでフラグメント化される場合があります（DF設定時はICMPで通知、PMTUDで調整）。

Q19. TLSはどの層に位置付けられますか？（実装上と機能上の両面で）

実装上はアプリケーション層で扱われることが多く、機能的にはプレゼンテーション層の役割（暗号化）。

Q20. トランスポート層の「多重化」とは何を使って実現しますか？

ポート番号を使って、複数のアプリケーション通信を同時に扱います。
