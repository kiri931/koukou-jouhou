TACHIHI BEACH NARA（ミ・ナーラ屋上）

人工ビーチでBBQができる屋上リゾート。砂浜・ステージ・コテージ風エリアなど非日常空間を体験。

  • 所在地：奈良市二条大路南1-3-1 屋上
  • BBQは2部制（10:00–15:00 / 16:00–21:00）
  • 食材持込OK、機材貸出・火起こし不要

出典：公式サイト

FAQ

服装は自由ですか？
動きやすい服装・靴で来てください。屋外エリアもあるため、天候に応じて羽織り物や帽子などを準備しましょう。
昼食はどうなりますか？
グループごとに予算内で購入し、屋上BBQエリアで飲食します。アレルギーがある場合は事前に教員へ連絡してください。
雨天の場合の対応は？
雨天でもスポッチャ・タチヒビーチは問題なく利用できます。
解散場所はどこですか？
基本はミ・ナーラ館内での解散となります。当日の指示で変更がある場合があります。
