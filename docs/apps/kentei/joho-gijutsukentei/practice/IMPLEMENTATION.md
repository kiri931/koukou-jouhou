# 分野別演習システム - 実装完了報告

## 📊 実装内容

NotebookLMで自動生成された4つのJSONファイルを統合し、**290問**の問題を扱える分野別演習システムを構築しました。

### データ構成

| ファイル | 問題数 | 形式 | 内容 |
|---------|--------|------|------|
| data/practice/mondai1.json | 130問 | 4択 | 基本用語（コンパイラ、OS、ネットワーク、著作権など） |
| data/practice/mondai2.json | 100問 | 4択 | 計算・論理回路 |
| data/practice/mondai3.json | 30問 | フローチャート | 穴埋め形式（基本計算、統計処理、物理計算など） |
| data/practice/mondai4.json | 30問 | プログラミング | C言語・BASIC（コード穴埋め） |

## 🎯 主な機能

### 1. 分野別（4分野）
- **基本用語** (mondai1.json)
- **計算・論理回路** (mondai2.json)
- **フローチャート** (mondai3.json)
- **プログラミング** (mondai4.json)

### 2. 問題表示形式

#### 選択式問題（mondai1.json, mondai2.json）
- 4択形式
- ラジオボタンで選択
- **自動採点対応** ✓

#### フローチャート問題（mondai3.json）
- フローチャートの流れを表示
- 穴埋め箇所を番号で表示
- 選択肢リストを表示
- 解答表示のみ（自動採点非対応）

#### プログラミング問題（mondai4.json）
- C言語とBASICをタブで切替
- コードエディタ風の表示
- 穴埋め箇所を①②③で表示
- 選択肢リストを表示
- 解答表示のみ（自動採点非対応）

### 3. 学習支援機能
- ✓ 分野ごとに約10問をランダム出題
- ✓ 問題の前後移動
- ✓ 回答の保存
- ✓ 採点機能（選択式のみ）
- ✓ 解説表示
- ✓ 進捗表示（何問目/全何問）

## 🔧 技術的特徴

### データローダー（practice-loader.js）
- 4つのJSONファイルを並列読み込み
- 各データ形式を統一フォーマットに変換
- カテゴリ自動分類（キーワードマッチング）
- エラーハンドリング（一部ファイル失敗でも動作）

### レンダリングエンジン（practice-engine.js）
- 問題形式に応じた動的レンダリング
- HTMLエスケープ処理
- フォーム処理と状態管理
- C言語/BASIC切替機能

### メインロジック（practice.js）
- 状態管理（state pattern）
- カテゴリ選択とフィルタリング
- シャッフル機能
- ナビゲーション制御

## 📝 使い方

1. **分野を選択**: ドロップダウンから学習したい分野を選択
2. **開始ボタンをクリック**: その分野から約10問がランダムに出題される
4. **回答を選択**: 選択式問題はラジオボタンで選択
5. **回答**: 入力すると自動で判定（正誤表示）
6. **解説を見る**: 解説を表示（正解した場合は自動表示）
8. **前へ/次へ**: 問題を移動
