# Cardgame (Static MVP)

- Open [index.html](index.html) in a browser.
- This is a local 2-player, turn-based card battle based on the spec.
- Turn switch overlay helps prevent peeking.

## Notes

- Cards are loaded from [cards.json](cards.json) (JSON comments are stripped on load).
- Decks are fixed in [app.js](app.js) for the MVP.
