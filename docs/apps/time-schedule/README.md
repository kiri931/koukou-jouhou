# Time Scheduler - タイムスケジューラ

視覚的に時間配分を管理できるタイムスケジューラアプリケーションです。

## 機能

### 📊 視覚的な円グラフ表示
- タスクごとの時間配分を時計のような円グラフで表示
- 現在の進行状況をリアルタイムで確認
- タスク間の境界線とラベル表示

### ⏱️ タイマー機能
- **スタート/一時停止/リセット** - 完全なタイマー制御
- **2つの開始モード**
  - 今から開始：現在時刻からスタート
  - 指定時刻から開始：過去の時刻を指定して進捗を計算
- 現在のタスク名、タスク時間、残り時間を大きく表示
- 現実世界の時刻も同時表示

### 📝 タスク管理
- タスクの追加・削除
- ドラッグ&ドロップで並び替え
- 各タスクに題目と時間を設定
- 未定時間の自動計算と表示

### 🎨 カスタマイズ
- **合計時間設定** - スライダーと数値入力の両方に対応
- **テーマカラー**
  - パステルカラーのプリセット6色
  - カスタムカラーの入力も可能
- **プリセット保存** - 設定をlocalStorageに保存して次回も利用可能

### 📱 レスポンシブ対応
- スマートフォンでも快適に利用可能
- タッチ操作に最適化

## 使い方

### 基本的な流れ

1. **設定ボタンをクリック**して設定パネルを開く
2. **合計時間**を設定（スライダーまたは数値入力）
3. **タスク**を追加・編集
   - タスク名と所要時間を入力
   - ドラッグで順番を変更
4. **開始モード**を選択
   - 今から開始 or 指定時刻から開始
5. **テーマカラー**を選択
6. **スタートボタン**でタイマー開始

### タスクの設定

- **追加**: 「➕ タスクを追加」ボタン
- **削除**: 各タスクの「✕」ボタン
- **並び替え**: ☰マークをドラッグ
- **編集**: タスク名と時間を直接入力

### タイマーの操作

- **▶️ スタート**: タイマー開始
- **⏸️ 一時停止**: タイマーを一時停止（再開可能）
- **🔄 リセット**: タイマーを最初に戻す

### 設定の保存

- **💾 現在の設定を保存**: 現在の設定をブラウザに保存
- **📂 保存した設定を読み込み**: 保存した設定を復元
- **🗑️ 保存データを削除**: 保存したデータを削除

## 技術仕様

- **HTML5 + CSS3 + Vanilla JavaScript**
- **SVG**による円グラフ描画
- **localStorage**によるデータ永続化
- **digital.css**デザインシステムを使用
- レスポンシブデザイン対応
