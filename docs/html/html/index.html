Webについて

HTML / CSS / JavaScript、タグと属性、ボックス、URL などの要点まとめ

Webページは何でできている？

  • HTML（内容と意味）
  • CSS（見た目・装飾）
  • JavaScript（動き・処理）

各ファイルの拡張子（保存形式）: .html / .css / .js

公開の仕組みとアドレス

外部に公開された Web サーバ（サーバ）に Web ページ情報を保存すると公開できる。ただし IP アドレス と URL（ドメイン名） が必要。

IP（Internet Protocol）

例: 192.168.0.1
世界中のインターネットにつながる端末に一つずつ割り振られている番号（大体）。

データベース

様々な情報を管理する Web ページでは、データベースが必要。

HTML（HyperText Markup Language）

文章に 意味や役割 を持たせるための言語。

HyperText
他の文章やページにジャンプできる仕組みを持つ。
Markup
タグと呼ばれる印を付けて、文章の役割を指定する。

基本のタグ例

<h1>見出し１</h1>  
<p>段落の文章</p>

タグに囲まれた部分を 要素（element）という。複数のタグで順番に記述していく → 入れ子構造。

HTML 文書の基本構造

<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>ページのタイトル</title>
  </head>
  <body>
    ページの内容
  </body>
</html>

<!DOCTYPE html> があると 標準準拠モード、ないと 互換モード で解釈される。

  • 標準準拠モード：最新の仕様に沿って表示
  • 互換モード：古いページとの互換を優先して表示

head と body の役割

html
HTML 文書全体を表す。
head
ページ自体の情報（メタデータの集合）。
body
ページの内容（ユーザーに見える部分）。
title
ページのタイトル（タブに表示）。
meta
ページに関する各種情報（例：文字コード）。

インデント（字下げ）を意識して、入れ子を見やすくしよう。

エレメント（要素）とは

HTMLでは、タグで囲まれた部分を 要素（エレメント） といいます。内容（コンテンツ）とタグで構成されています。

<タグ名>内容</タグ名>

代表的なエレメントの例：

タグ 意味
a リンク
abbr 略語
br 改行
button ボタン
div 意味を持たないブロック
em 強調（斜体）
h1 見出し1
iframe ほかのサイトの一部を表示
img 画像
input 入力欄
script JavaScriptを記述
span 意味を持たないインライン要素
strong 強調（太字）
u アンダーライン
ul 順序なしリスト
ol 順序ありリスト
li リスト項目

div と span の違い

  • div：ブロック要素（前後に改行が入る）
  • span：インライン要素（前後に改行が入らない）
体験用ページ

属性（attributes）

要素に設定や付加情報を追加できる。

<html lang="ja">  
<!-- 複数属性はスペース区切り -->
<要素 属性名1="値1" 属性名2="値2">...</要素>

グローバル属性（どの要素にも付けられる）

  • class：分類名（CSS の対象にできる）
  • id：固有名
  • lang：言語（例：ja、en）
  • style：個別の CSS を直接指定
  • title：補足情報（ツールチップなど）
<div class="kyocho">赤くなるよ</div>
/**** CSS 例 ****/
.kyocho { color: red; }

メタ要素

ページの様々な情報を設定する。

<meta charset="UTF-8" />
  • UTF-8（Unicode）…世界中の文字
  • Shift_JIS（Windows 用）
  • EUC-JP（UNIX 用）

要素の配置ルールとカテゴリ

代表例（抜粋）：

  • Metadata content：meta / script / style / title
  • Flow content：body 内で使う多くの要素（a / abbr / br / button / div / em / h1 / iframe / img / input / script / span / strong / u / ul / ol / li など）
  • Heading content：見出し（h1〜h6）
  • Phrasing content：テキストに意味（a / abbr / br / em など）
  • Embedded content：埋め込み（iframe / img）
  • Interactive content：操作できる要素（a / button / iframe / img / input）
  • Sectioning content：章・節・項など話題の範囲

div と span の違い

  • div：ブロック要素（前後に改行）
  • span：インライン要素（前後に改行しない）

特殊な記述（エスケープ）

記号 エスケープ
< &lt;
> &gt;
& &amp;
" &quot;

例

<!-- 誤 -->
<p>こんにちは<私は足立>です。</p>

<!-- 正 -->
<p>こんにちは&lt;私は足立&gt;です。</p>

Web ブラウザと開発者ツール

代表例：IE / Chrome / Firefox / Safari / Edge

F12 で開発者ツールを起動できる。

  • Elements：HTML・CSS の確認と編集
  • Console：エラーメッセージ表示・JS 実行

ボックスモデル

ブラウザは HTML 要素を長方形の「ボックス」として配置する。

  • コンテンツ（内容）
  • パディング（内側の余白）
  • ボーダー（枠線）
  • マージン（外側の余白）

CSS での主な指定

  • width / height
  • padding / border / margin

ブロック要素とインライン要素の性質の違いにも注意。

ハイパーリンク（リンク）

クリックで別の Web ページを表示。

URL 例：http://example.com/sample/index.html

  • http：プロトコル（約束事）
  • example.com：ドメイン名
  • sample：ディレクトリ名
  • .html：拡張子

Flexbox（Flexible Box Layout Module）

/* 親要素に指定 */
.container { display: flex; }

CSS で覚えておきたいプロパティ

  • font-size（文字の大きさ） / font-weight（太さ） / text-align（left/center/right）
  • color（文字色） / background-color（背景色） / background-image（背景画像）
  • width（横幅） / height（縦幅）
  • border（外枠）
