Webってなに？

Webページは、インターネットで見られるページのことです。

Webページを作るには、次の3つのファイルを使います：

  • HTML（.html）：ページの内容や構造を決める
  • CSS（.css）：色やデザインなど見た目を決める
  • JavaScript（.js）：動きをつける

作ったページをみんなに見せるには「Webサーバー」に置きます。

Webサーバーには「IPアドレス（例：192.168.0.1）」や「URL（例：https://example.com）」が必要です。

たくさんのデータを管理したいときは「データベース」を使います。

HTMLとは？

HTML（エイチティーエムエル）は、Webページの「骨組み」を作るための言語です。

  • HyperText：リンクで他のページに移動できる
  • Markup：タグ（<h1>など）で文章の役割を示す

たとえば、見出しは次のように書きます：

<h1>見出し</h1>

HTMLは「タグ」で囲んで意味を伝えます。

CSSとは？

CSS（シーエスエス）は、Webページの色や文字の大きさ、レイアウトなど「見た目」を決めるための言語です。

たとえば、文字を赤くしたいときは次のように書きます：

.kyocho {
    color: red;
}

このように、CSSを使うと自分好みのデザインにできます。

JavaScriptとは

JavaScriptは、Webページに「動き」や「反応」を加えるためのプログラミング言語です。

  • 画像が自動で切り替わるスライドショー
  • スクロールするとアニメーションするページ
  • ボタンを押すとメニューが開く

※「Java」と名前が似ていますが、まったく別の言語です。

HTMLの基本構造

HTMLファイルは、次のような決まった形（ひな形）で書き始めます。

<!DOCTYPE html>
<html>
<head>
    <title>ページのタイトル</title>
</head>
<body>
    ページの内容
</body>
</html>
  • <!DOCTYPE html>：HTML5で書くことを宣言します。
  • <html> ～ </html>：HTML文書の始まりと終わりを示します。
  • <head> ～ </head>：ページの情報（タイトルや文字コードなど）を書きます。
  • <body> ～ </body>：実際に画面に表示される内容を書きます。

インデント（字下げ）を使うと、どこからどこまでがセットなのか分かりやすくなります。

属性とグローバル属性

属性（attribute）は、タグに「追加の情報」をつけるためのものです。

たとえば、ページの言語を指定するには次のように書きます：

<html lang="ja">

この lang="ja" の部分が「属性」です。

どのタグでも使える「グローバル属性」には、次のようなものがあります：

  • class：CSSでデザインを変えたいときに使う名前
  • id：ページ内で1つだけの特別な名前
  • lang：その部分の言語を指定
  • style：その場だけCSSを書きたいときに使う
  • title：マウスを重ねると説明が表示される

属性は、タグ名 属性名="値" の形で書きます。

要素の分類とカテゴリ

HTMLのタグ（要素）は、役割ごとに7つのグループ（カテゴリ）に分かれています。

  • メタデータ・コンテンツ：ページの情報（例：<title>や<meta>）
  • フロー・コンテンツ：ページの主な流れを作る要素（例：<p>や<div>）
  • 見出しコンテンツ：見出しを表す要素（例：<h1>～<h6>）
  • フレージング・コンテンツ：文章やその一部を表す要素（例：<span>や<strong>）
  • 埋め込みコンテンツ：画像や動画など、他のデータを埋め込む要素（例：<img>や<video>）
  • インタラクティブ・コンテンツ：ユーザーが操作できる要素（例：<a>や<button>）
  • セクショニング・コンテンツ：ページを区切る要素（例：<section>や<nav>）

このように、タグの役割ごとにグループ分けされているので、どんな場面で使うかが分かりやすくなっています。

特殊文字の書き方

HTMLでは、「<」や「>」などの記号をそのまま書くと、タグと間違われて正しく表示されません。
そのため、特別な書き方（エスケープ）を使います。

  • <（小なり）は &lt; と書きます
  • >（大なり）は &gt; と書きます
  • &（アンド）は &amp; と書きます
  • "（ダブルクォーテーション）は &quot; と書きます

たとえば、次のように書くと…

<p>こんにちは&lt;私は足立&gt;です。</p>

画面には「こんにちは<私は足立>です。」と表示されます。

Webブラウザと開発者ツール

Webページを見るときは「Webブラウザ」（例：Chrome、Firefox、Edge、Safariなど）を使います。
Webブラウザには「開発者ツール」という便利な機能があり、F12キーを押すと開くことができます。

開発者ツールでは、次のようなことができます：

  • Elements（エレメンツ）：ページのHTMLやCSSを確認したり、その場で編集できます。
  • Console（コンソール）：JavaScriptを実行したり、エラーやメッセージを確認できます。

これらを使うと、Webページの仕組みを調べたり、作ったページの動きをチェックしたりできます。
初めての人も、いろいろ触ってみると理解が深まります。

ボックスモデル

Webページのパーツ（文字や画像など）は、すべて「四角い箱（ボックス）」として考えます。
この箱は、次の4つの部分でできています：

  • content（内容）：文字や画像など、実際に表示される部分
  • padding（内側の余白）：内容のまわりのスペース。背景色もここまで広がります
  • border（枠線）：箱のふちを囲む線
  • margin（外側の余白）：他の箱との間をあけるスペース

たとえば、プレゼントの箱をイメージすると、
内容がプレゼント、paddingがクッション、borderが箱のふち、marginが他の箱とのすき間です。

CSSの基本プロパティ

CSSには、よく使う「プロパティ（設定項目）」がいくつかあります。
これらを使うと、文字の大きさや色、背景などを自由に変えられます。

  • width / height：箱（要素）の横幅や高さを決めます
  • font-size / font-weight：文字の大きさや太さを変えます
  • text-align：文字を左・中央・右にそろえます（例：left/center/right）
  • color：文字の色を指定します
  • background-color / background-image：背景の色や画像を設定します
  • border：枠線をつけます

たとえば、「文字を赤くして大きくしたい」「背景に色をつけたい」といったときに、これらのプロパティを使います。

Flexboxについて

Flexbox（フレックスボックス）は、Webページのレイアウトを簡単にきれいに並べるための仕組みです。
たとえば、写真やボタンを横に並べたいとき、Flexboxを使うととても便利です。

まず、親要素に display: flex; を指定します。すると、その中にある子要素が自動的に横並びになります。 

.container {
    display: flex;
}

これだけで、.container の中の要素が横に並びます。
他にも、justify-content や align-items などを使うと、間隔や位置も自由に調整できます。

  • justify-content：子要素の横方向の並び方を決める（例：中央寄せ、右寄せなど）
  • align-items：子要素の縦方向の位置を決める

Flexboxを使うと、難しい計算をしなくても、思い通りに要素を並べることができます。
