シューティングゲーム制作学習

HTML / CSS / JavaScript を用いて簡単なシューティングゲームを作っていきます。
読みながら進めることで、ゲームができていきます。

このサイトについて

このサイトでは、プログラミング初心者の人でも
シューティングゲームを一から作れるようになることを目標にしています。

「動いた！」「当たった！」を大切にしながら、
少しずつコードを書いていきます。

学習の流れ

  1. ゲーム画面を表示してみる
  2. 自機（プレイヤー）を動かす
  3. 弾を発射する
  4. 敵を出現させる
  5. 当たり判定を作る
  6. スコアやライフを追加する

さっそく始める

まずは「ゲーム画面を表示する」ところから始めましょう。
